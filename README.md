# New Skill Workspace

This repository contains various skills and tools for enhancing development workflows.

## Structure

- **oneshot-llm-docs/**: Contains documentation and scripts for LLM-based skills.
- **oneshot-pdf-docs/**: Includes tools for processing and packaging PDF documents.
- **python/**: Python-related skills and best practices.
- **scripts/**: Utility scripts for various tasks.

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/scrossle-teck/new-skill-workspace.git
   ```

2. Navigate to the project directory:

   ```bash
   cd new-skill-workspace
   ```

3. Set up the Python environment:

   ```bash
   python -m venv .venv
   .\.venv\Scripts\activate
   pip install -r requirements.txt
   ```
